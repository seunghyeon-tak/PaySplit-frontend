# PaySplit Frontend

구독 결제 관리 시스템 프론트엔드

## 기술 스택

- React 19
- TypeScript
- Vite

## 시작하기

\```bash
# 패키지 설치
npm install

# 개발 서버 실행
npm run dev
\```

## 프로젝트 구조

```
src/
├── assets/          # 이미지, 폰트 등
├── components/      # 재사용 가능한 UI 조각들
├── pages/           # 화면 단위 컴포넌트
├── hooks/           # 커스텀 훅
├── services/        # API 호출 관련
├── types/           # TypeScript 타입 정의
├── utils/           # 공통 유틸 함수
└── styles/          # 전역 스타일
```
